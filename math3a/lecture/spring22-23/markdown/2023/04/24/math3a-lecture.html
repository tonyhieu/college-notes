Linear functions

A function takes elements from one set (known as the domain) and outputs elements of another set (known as the range or codomain).

Multiplying by a m * n matrix (where m is the number of columns and n is the number of rows) defines a function that maps elements from Rn to Rm.

T:Rn→Rm is linear ifT(v→+w→)=T(v→)+T(w→) and T(λv→)=λT(v→)T: \mathbb{R}^n \rightarrow \mathbb{R}^m \text{ is linear if} \\ T(\overrightarrow{v} + \overrightarrow{w}) = T(\overrightarrow{v}) + T(\overrightarrow{w}) \text{ and } T(\lambda \overrightarrow{v}) = \lambda T(\overrightarrow{v})T:Rn→Rm is linear ifT(v+w)=T(v)+T(w) and T(λv)=λT(v)
